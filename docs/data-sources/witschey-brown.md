# Witschey / Brown data source note

## Initial source used in Maya Atlas
We are starting with the public Witschey/Brown-derived dataset hosted on Data Basin.

Dataset page:
- "Mayan Sites / Ruinas Maya"

Notes from source page:
- Includes Maya sites in and near the Maya region
- Rank 1 (very large), Rank 2 (large), and Rank 3–5 (small to very small)
- Extracted from the more than 6,000 archaeological sites in the Electronic Atlas of Ancient Maya Sites
- Source page states CC BY 3.0

## Use in Maya Atlas
This source is currently used only for:
- site corpus ingestion
- site name normalization
- coordinates
- rank-driven major site flagging

This source is not yet being used for:
- chronology
- population
- political regions
- detailed media or bibliography expansion

## Citation
Walter R.T. Witschey and Clifford T. Brown, via Data Basin public dataset.
